## 🚀 使用方法

### 创建新文章

1. 进入"博客"页面
2. 点击"发布文章"按钮
3. 填写文章信息：
   - 封面图片地址（可选）
   - 文章标题
   - 文章简介
   - 选择分类
   - 输入主密码：`jzh0128`
4. 点击"编辑正文"按钮
5. 使用富文本编辑器编写内容
6. 点击"💾 保存并同步"

### 编辑现有文章

1. 在文章列表中点击"编辑"按钮
2. 或在文章详情页点击"编辑"按钮
3. 在富文本编辑器中修改内容
4. 点击"💾 保存并同步"

### 插入图片

**方法一：使用工具栏**
1. 点击工具栏的图片图标 🖼️
2. 输入图片 URL
3. 点击确定

**方法二：直接粘贴**
- 从网页复制图片后，直接在编辑器中粘贴（部分浏览器支持）

### 插入链接

1. 选中要添加链接的文字
2. 点击工具栏的链接图标 🔗
3. 输入链接地址
4. 点击确定

### 同步到 GitHub

1. 在编辑页面输入您的 GitHub Personal Access Token
2. 点击"💾 保存并同步"
3. 文章会自动上传到您的 GitHub 仓库对应分类目录

## 💡 使用技巧

### 1. 快速格式化
- 选中文字后，点击工具栏按钮即可应用格式
- 可以同时应用多种格式（如粗体+斜体+颜色）

### 2. 复制粘贴
- 从 Word、网页等地方复制的内容会保留格式
- 如果不想保留格式，先粘贴到记事本，再复制到编辑器

### 3. 撤销/重做
- 使用浏览器的快捷键：
  - 撤销：`Ctrl+Z` (Windows) 或 `Cmd+Z` (Mac)
  - 重做：`Ctrl+Y` (Windows) 或 `Cmd+Shift+Z` (Mac)

### 4. 图片建议
- 使用图床服务（如 GitHub、图床网站）上传图片
- 获取图片的直链 URL
- 在编辑器中插入图片 URL

### 5. 保存习惯
- 编写长文时，定期点击"保存"
- 浏览器会自动保存到本地
- 使用 GitHub Token 可以同步到云端

## 🔧 常见问题

### Q: 编辑器没有加载怎么办？
**A:** 刷新页面，确保网络连接正常，Quill.js 库会从 CDN 加载。

### Q: 如何更改文字颜色？
**A:** 选中文字 → 点击工具栏的颜色按钮 → 选择颜色。

### Q: 如何插入本地图片？
**A:** 需要先将图片上传到图床或 GitHub，获取 URL 后插入。

### Q: 保存的内容在哪里？
**A:** 
- 本地：浏览器的 localStorage
- 远程：GitHub 仓库（如果提供了 Token）

### Q: 如何导出文章？
**A:** 文章以 HTML 格式保存，可以：
1. 复制编辑器中的内容到 Word
2. 使用浏览器的"打印"功能保存为 PDF
3. 同步到 GitHub 后下载 HTML 文件
