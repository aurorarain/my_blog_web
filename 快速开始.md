# 🚀 快速开始指南

## 📝 创建并同步文章（完整流程）

### 1️⃣ 准备 GitHub Token
```
1. 访问：https://github.com/settings/tokens
2. 点击：Generate new token (classic)
3. 勾选：repo (完整权限)
4. 生成并复制 Token
```

### 2️⃣ 创建文章
```
博客 → 发布文章
├─ 封面URL: https://example.com/image.jpg (可选)
├─ 本地封面: 选择文件 (可选)
├─ 标题: 我的第一篇文章
├─ 简介: 这是文章简介
├─ 分类: 随笔
├─ ☑️ 发布到 GitHub 仓库
├─ GitHub Token: ghp_xxxxxxxxxxxx
└─ 密码: jzh0128
```

### 3️⃣ 编写内容
```
点击"编辑正文" → 使用富文本编辑器编写
```

### 4️⃣ 保存并同步
```
输入 GitHub Token → 点击"💾 保存并同步"
```

## 💡 快速提示

### 封面图片
- **URL方式**: 直接输入图片链接
- **本地上传**: 选择文件 + 勾选同步 + 输入Token

### 同步时机
- **创建时**: 勾选"发布到 GitHub 仓库"
- **编辑内容**: 输入 Token 后点击"保存并同步"
- **修改元数据**: 勾选同步 + 输入 Token，系统会更新 GitHub 文件路径
- **删除文章**: 输入 Token 可同步删除 GitHub 文件

### 修改标题或分类
- 如果文章已同步到 GitHub
- 修改标题/分类后，勾选"发布到 GitHub 仓库"并输入 Token
- 系统会删除旧路径文件，在新路径创建文件

### 删除文章
- **编辑页面删除**: 点击"🗑️ 删除" → 输入 Token
- **列表删除**: 点击"删除" → 输入主密码 → 输入 Token
- 提供 Token 可同步删除 GitHub 文件，否则仅删除本地

### Token 安全
- ⚠️ 不要分享
- ⚠️ 不要提交到代码
- ✅ 定期更换
