# 个人博客（静态模版）

快速说明：这是一个简单的静态个人博客模版，支持中/英切换、四个页面（首页、项目、分类、留言板）和统一背景图配置。编辑 `app.js` 顶部的配置项即可替换你的照片、简介和背景图地址。

配置位置：

- 打开 `app.js`，修改 `BG_IMAGE` 为你的背景图地址（字符串）。
- 修改 `USER_PHOTO`、`USER_NAME`、`USER_BIO`、`USER_CONTACT` 为你的信息。

本地预览：直接用文件浏览器打开 `index.html` 或用简单静态服务器（例如 VS Code Live Server 插件）。

留言板：使用 `localStorage` 保存留言，无需登录。留言数据保存在浏览器，本地清除或更换浏览器会丢失数据。

进一步扩展建议：
- 将数据改为从后端 API 获取（项目、文章、留言）
- 添加 Markdown 渲染与文章分页
- 增强无障碍支持与 SEO 元信息
