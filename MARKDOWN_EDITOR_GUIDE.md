# Markdown 编辑器使用指南

## 📝 编辑器功能说明

您的博客现在使用 **EasyMDE** 作为 Markdown 编辑器，它提供了丰富的功能和良好的用户体验。

### ✨ 主要功能

#### 1. **工具栏功能**
- **粗体** (Ctrl+B): 将选中文字加粗
- **斜体** (Ctrl+I): 将选中文字倾斜
- **删除线**: 在文字上添加删除线
- **标题**: 一级、二级、三级标题
- **引用**: 创建引用块
- **列表**: 无序列表和有序列表
- **代码块** (Ctrl+Alt+C): 插入代码块
- **链接** (Ctrl+K): 插入超链接
- **图片**: 插入图片
- **表格**: 插入表格

#### 2. **预览模式**
- **预览** (Ctrl+P): 切换预览模式
- **分屏预览** (F9): 左侧编辑，右侧实时预览
- **全屏** (F11): 全屏编辑模式

#### 3. **编辑辅助**
- **撤销** (Ctrl+Z): 撤销上一步操作
- **重做** (Ctrl+Y): 重做操作
- **自动保存**: 每秒自动保存到浏览器本地
- **状态栏**: 显示行数、字数、光标位置

### 📖 Markdown 语法快速参考

#### 标题
```markdown
# 一级标题
## 二级标题
### 三级标题
```

#### 文字样式
```markdown
**粗体文字**
*斜体文字*
~~删除线~~
`行内代码`
```

#### 列表
```markdown
- 无序列表项 1
- 无序列表项 2

1. 有序列表项 1
2. 有序列表项 2
```

#### 链接和图片
```markdown
[链接文字](https://example.com)
![图片描述](https://example.com/image.jpg)
```

#### 引用
```markdown
> 这是一段引用文字
> 可以多行
```

#### 代码块
````markdown
```javascript
function hello() {
    console.log("Hello World!");
}
```
````

#### 表格
```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |
```

#### 分隔线
```markdown
---
```

### 🎯 使用技巧

1. **快捷键**: 熟练使用快捷键可以大大提高编辑效率
2. **分屏预览**: 使用 F9 开启分屏，边写边看效果
3. **自动保存**: 编辑器会自动保存，不用担心丢失内容
4. **GitHub 同步**: 填写 GitHub Token 可以将文章同步到远程仓库

### 🔧 高级功能

#### GitHub 远程同步
1. 在编辑页面输入您的 GitHub Personal Access Token
2. 点击"保存并同步"按钮
3. 文章会自动上传到您的 GitHub 仓库对应分类目录

#### 文章管理
- **编辑**: 点击文章卡片上的"编辑"按钮
- **删除**: 在编辑页面点击"删除"按钮（需确认）
- **设置**: 点击"⚙️ 设置"修改文章标题、简介、分类等元数据

### 📱 移动端支持

编辑器在移动设备上也能正常使用，但建议在桌面端进行长文编辑以获得最佳体验。

### 🆘 常见问题

**Q: 编辑器没有加载怎么办？**
A: 检查网络连接，确保 CDN 资源能正常加载。

**Q: 自动保存的内容在哪里？**
A: 保存在浏览器的 localStorage 中，清除浏览器数据会丢失。

**Q: 如何备份文章？**
A: 使用 GitHub 同步功能，或者手动复制 Markdown 内容。

**Q: 支持哪些 Markdown 语法？**
A: 支持 GitHub Flavored Markdown (GFM) 的所有语法。

### 🎨 自定义样式

编辑器的样式可以通过修改 `styles.css` 中的 `.EasyMDEContainer` 相关样式来自定义。
